Contact the NHS Pensions

Use this service if you need to contact NHS Pensions for any pensions queries.

You can use this service if you need help with:

  • the NHS pension scheme, 1995/2008 or 2015
  • my NHS pension (MNP)
  • your total reward statement (TRS)
  • anything related to NHS Pensions

Before you start

If you are a member, we'll ask you for your:

  • membership number if you know it
  • date of birth
  • first and last name
  • address

If you are an employer, we'll ask you for your:

  • employing authority code

If you are an employer contacting on behalf of a member, we'll ask for the member's details, including their membership number.
