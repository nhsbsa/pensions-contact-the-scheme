An EA code identifies each employer in the NHS Pension Scheme.

our EA code should be between 4 and 6 characters long and it can be:

  • 4 numbers like 1234
  • 1 letter followed by 3 numbers like A123
  • 2 letters followed by 4 numbers like EA1234
  • 3 letters followed by 3 numbers like EAA123

If you need to find your EA code, use the employing authority lookup tool. You will need to enter either your:

  • organisations name
  • organisations postcode.
