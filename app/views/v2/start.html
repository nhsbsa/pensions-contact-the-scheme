Use this service if you need to contact NHS Pensions for any pensions queries.

You can use this service if you are a member, employer or a third-party authority and need help with:

  • the NHS Pension Scheme (1995/2008 or 2015)
  • the My NHS Pension Portal
  • your Total Reward Statement (TRS)
  • anything relating to NHS Pensions

Before you start

If you are a member, we'll ask you for your:

  • membership number (if you know it)
  • date of birth
  • first and last name
  • address

If you are an employer, we'll ask you for your:

  • employing authority code

If you are an employer contacting on behalf of a member, we'll ask for the member's:

  • membership number
  • date of birth
  • first and last names
  • address

If you are a third-party authority, we'll ask you for your:

  • first and last names
  • contact details

We'll ask you for the member's:

  • membership number
  • date of birth
  • first and last names
  • address
