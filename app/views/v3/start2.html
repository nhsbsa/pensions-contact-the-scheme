Use this service if you need to contact NHS Pensions for any pensions queries.

You can use this service if you are a member, employer or a third-party authority and need help with:

  • the NHS Pension Scheme (1995/2008 or 2015)
  • the My NHS Pension Portal
  • your Total Reward Statement (TRS)
  • anything relating to NHS Pensions

Before you start

Members

If you’re a member, we'll ask for your:

  • membership number
  • personal details (name, address, date of birth)

Employers

If you’re an employer, we'll ask for your employing authority (EA) code.

We’ll also ask for the member’s:

  • personal details
  • membership number

Acting for a member

If you’re acting for a member, we'll ask for your name and contact details.

We’ll also ask for the member’s:

  • personal details
  • membership number
